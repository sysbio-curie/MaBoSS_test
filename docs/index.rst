MaBoSS_test
===========

MaBoSS_test is model testing library for models built with the `MaBoSS <https://maboss.curie.fr/>`_ software.
It is based on python unittest standard library, and follow the same coding structure.
The library can be loaded by running the following line in Python::
	import maboss_test
